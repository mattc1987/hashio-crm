✨ Hashio CRM — extension settings

Connect this extension to your Apps Script backend. Same URL + API key the web app uses.

The deployed Apps Script URL (ends in /exec). Find this in Apps Script → Deploy → Manage deployments.
Your Apps Script API key — same one stored as VITE_APPS_SCRIPT_KEY in your .env file.
Where do I find these?
Both values are already in your Hashio CRM project. Open the file .env in the repo root — copy the VITE_APPS_SCRIPT_URL and VITE_APPS_SCRIPT_KEY values directly. They're already deployed and working with the web app.

Once saved, click the Hashio CRM icon in your Chrome toolbar while viewing a Gmail email. The popup will show the contact, recent activity, and quick actions (log email, create task, create deal, AI BDR).
